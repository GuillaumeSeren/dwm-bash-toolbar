dwm-bash-toolbar
================
Simple bash toolbar for DWM

## Why ?
Getting a light toolbar for DWM

## Features
Here the main features of the script:

MODE     | DESCRIPTION
---------|------------
`WiFi`   | The wifi interface
`Lang`   | The keyboard layout
`Power`  | Battery information
`Vol`    | Sound volume information
`Date`   | Date and time
